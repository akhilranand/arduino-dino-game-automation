# Dino Game Automation

An Arduino-based automation project designed to play the Chrome Dino game automatically using sensor input and servo control.

## Features
- Detects game conditions using sensor input
- Automates jump action in the Dino game
- Uses servo motor for physical interaction
- Simple robotics + automation project

## Components Used
- Arduino
- Servo Motor
- Sensor / LDR
- LEDs
- Jumper Wires
- Power Supply

## File
- `dinoGameAutomation.ino` – Arduino source code
